# LittleLemon

LittleLemon is a native iOS food ordering app built using SwiftUI. This project is part of the Coursera Meta iOS Certificate program.

## Certificate Achievements
- **Introduction to iOS Mobile Application Development:** [Verify Certificate](https://coursera.org/verify/YUVDQH5F2AH8)
- **Programming Fundamentals in Swift:** [Verify Certificate](https://coursera.org/verify/5GP5E7SZ2MGQ)
- **Advanced Programming in Swift:** [Verify Certificate](https://coursera.org/verify/Y2WKGQMLHZQV)
- **Create the User Interface with SwiftUI:** [Verify Certificate](https://coursera.org/verify/DAAR8B3BPXGS)
- **Working with Data in iOS:** [Verify Certificate](https://coursera.org/verify/MQA48HY8Y85P)
- **iOS App Capstone:** [Verify Certificate](https://coursera.org/verify/5X7JXWPGTXSN)
- **Version Control:** [Verify Certificate](https://coursera.org/verify/7N2MS3LMQXZZ)
- **Coding Interview Preparation:** [Verify Certificate](https://coursera.org/verify/3KNY7Y2PHKLW)
